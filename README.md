# Frontend assignment Schiphol

The app is deployed on Vercel and visible on:
[Live Demo](https://frontend-assignment-schiphol.vercel.app/)


# How to install

```
yarn install
```

# How to run

```
yarn dev
yarn install
```

# How to build and start the build

```
yarn build
yarn preview
```

# How to test 

```
yarn test
```
